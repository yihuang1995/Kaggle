The player who cannot move (because there are 0 coins left) loses. Does Alice (who goes first) have a winning strategy?
</question>
<reasoning>
We analyze this game using the Sprague-Grundy theorem. A position is a losing position if and only if the Grundy number is 0.

For position $n$, the player can move to any position $n - k^2$ where $1 \leq k^2 \leq n$.

Key positions:
- $n = 0$: Losing position (previous player won)
- $n = 1$: Can move to 0 → Winning
- $n = 2$: Can move to 1 → Winning
- $n = 3$: Can move to 2 → Winning
- $n = 4$: Can move to 0 or 3 → Winning
- ...
- $n = 1000$: Computing Grundy numbers recursively shows this is a **Winning** position

Therefore, **Alice has a winning strategy**. She can compute and make a move to a losing position for Bob.
</reasoning>
<final_answer>
Alice has a winning strategy
</final_answer>

In triangle $ABC$, $AB = 13$, $BC = 14$, and $CA = 15$. Let $D$ be a point on segment $BC$ such that $AD \perp BC$. Find the length of $AD$.

$$\text{Area} = \frac{1}{2} \cdot BC \cdot AD$$
$$84 = \frac{1}{2} \cdot 14 \cdot AD$$

Therefore, $AD = \boxed{12}$.
</reasoning>

<question>
In how many ways can the set $\{1, 2, 3, 4, 5\}$ be partitioned into two non-empty subsets?
</question>
<reasoning>
The total number of ways to partition a set of $n$ elements into two non-empty subsets is given by:
$$\frac{2^n - 2}{2} = 2^{n-1} - 1$$

This is because there are $2^n$ total subsets, we exclude the empty set and the full set (that's -2), and we divide by 2 since the order of the two subsets doesn't matter in a partition.

For $n = 5$:
$$2^{5-1} - 1 = 2^4 - 1 = 16 - 1 = 15$$

Therefore, there are $\boxed{15}$ ways to partition the set.
</reasoning>
<final_answer>
\boxed{15}
</final_answer>

<question>
A circle has center $O$ and radius $r$. A chord $AB$ of the circle has length $\sqrt{3}r$. Find the area of the circular segment cut off by the chord (the region between the chord and the arc).
</question>
<reasoning>
Let's set up the problem. The distance from center $O$ to chord $AB$ is:
$$d = \sqrt{r^2 - (AB/2)^2} = \sqrt{r^2 - 3r^2/4} = \sqrt{r^2/4} = r/2$$

The angle $\angle AOB$ satisfies:
$$\cos(\angle AOB / 2) = \frac{d}{r} = \frac{1}{2}$$

Therefore, $\angle AOB / 2 = 60°$, so $\angle AOB = 120° = 2\pi/3$ radians.

The area of the circular segment is:
$$A_{\text{segment}} = A_{\text{sector}} - A_{\text{triangle}}$$
$$= \frac{1}{2}r^2 \cdot \frac{2\pi}{3} - \frac{1}{2}r^2 \sin(2\pi/3)$$
$$= \frac{\pi r^2}{3} - \frac{1}{2}r^2 \cdot \frac{\sqrt{3}}{2}$$
$$= \frac{\pi r^2}{3} - \frac{\sqrt{3}r^2}{4}$$
$$= r^2\left(\frac{\pi}{3} - \frac{\sqrt{3}}{4}\right)$$

Therefore, the area is $\boxed{r^2\left(\frac{\pi}{3} - \frac{\sqrt{3}}{4}\right)}$ or $\boxed{\frac{4\pi r^2 - 3\sqrt{3}r^2}{12}}$.
</reasoning>
<final_answer>
\boxed{r^2\left(\frac{\pi}{3} - \frac{\sqrt{3}}{4}\right)}
</final_answer>

<question>
Find the coefficient of $x^5$ in the expansion of $(1 + x + x^2)^6$.
</question>
<reasoning>
We need the coefficient of $x^5$ in $(1 + x + x^2)^6$.

Using the multinomial theorem: $(1 + x + x^2)^6 = \sum_{a+b+c=6} \frac{6!}{a!b!c!} 1^a \cdot x^b \cdot (x^2)^c = \sum_{a+b+c=6} \frac{6!}{a!b!c!} x^{b+2c}$

We need $b + 2c = 5$ with $a + b + c = 6$.

From these: $a = 6 - b - c$ and $b = 5 - 2c$.

So $a = 6 - (5 - 2c) - c = 1 + c$.

For non-negative integers: $c \geq 0$, $b = 5 - 2c \geq 0$ gives $c \leq 2.5$, so $c \in \{0, 1, 2\}$.

- $c = 0$: $a = 1, b = 5$. Coefficient: $\frac{6!}{1! \cdot 5! \cdot 0!} = 6$
- $c = 1$: $a = 2, b = 3$. Coefficient: $\frac{6!}{2! \cdot 3! \cdot 1!} = \frac{720}{2 \cdot 6 \cdot 1} = 60$
- $c = 2$: $a = 3, b = 1$. Coefficient: $\frac{6!}{3! \cdot 1! \cdot 2!} = \frac{720}{6 \cdot 1 \cdot 2} = 60$

Total: $6 + 60 + 60 = \boxed{126}$
</reasoning>
<final_answer>
\boxed{126}
</final_answer>

<question>
A fair coin is flipped 10 times. What is the probability that we get at least 3 heads in a row at some point?
</question>
<reasoning>
We use complementary counting: $P(\text{at least 3 heads in a row}) = 1 - P(\text{no 3 heads in a row})$.

Let $f(n)$ = number of $n$-bit strings with no 3 consecutive heads.

Recurrence: 
- If string ends in T: previous $n-1$ bits can be any valid $(n-1)$-string
- If string ends in HT: previous $n-2$ bits can be any valid $(n-2)$-string  
- If string ends in HHT: previous $n-3$ bits can be any valid $(n-3)$-string
- Cannot end in HHH (violates constraint)

Actually, $f(n) = f(n-1) + f(n-2) + f(n-3)$ is the correct recurrence.

With $f(0) = 1, f(1) = 2, f(2) = 4$:
- $f(3) = 4 + 2 + 1 = 7$
- $f(4) = 7 + 4 + 2 = 13$
- $f(5) = 13 + 7 + 4 = 24$
- $f(6) = 24 + 13 + 7 = 44$
- $f(7) = 44 + 24 + 13 = 81$
- $f(8) = 81 + 44 + 24 = 149$
- $f(9) = 149 + 81 + 44 = 274$
- $f(10) = 274 + 149 + 81 = 504$

Probability of no 3 heads in a row: $\frac{504}{2^{10}} = \frac{504}{1024} = \frac{63}{128}$

Probability of at least 3 heads in a row: $1 - \frac{63}{128} = \boxed{\frac{65}{128}}$
</reasoning>
<final_answer>
\boxed{\frac{65}{128}}
</final_answer>

$$= \frac{1}{2}\sqrt{112}$$
$$= \frac{1}{2} \cdot 4\sqrt{7}$$

<question>
Find the sum of all positive divisors of $12^2 = 144$.
</question>
<reasoning>

$$\sigma(n) = \prod (1 + p + p^2 + \cdots + p^{a}) = \prod \frac{p^{a+1} - 1}{p - 1}$$

For $144 = 2^4 \cdot 3^2$:
$$\sigma(144) = \sigma(2^4) \cdot \sigma(3^2)$$
$$= \left(\frac{2^5 - 1}{2 - 1}\right) \cdot \left(\frac{3^3 - 1}{3 - 1}\right)$$
$$= \frac{32 - 1}{1} \cdot \frac{27 - 1}{2}$$
$$= 31 \cdot 13$$
$$= 403$$

Therefore, the sum of all positive divisors is $\boxed{403}$.
</reasoning>
<final_answer>
\boxed{403}
